# 貢献ガイド

Autonomous Airpower Watchへの情報追加、訂正、情報源の提案を歓迎します。

## ニュースの追加

掲載データは`data/news.json`へ追加します。必須項目と判断基準は[CLAUDE.md](./CLAUDE.md)に定義しています。

- 原文を確認し、タイトルだけで分類しないでください。
- 一次情報を優先してください。
- 発表、選定、契約、試験、量産、配備を区別してください。
- 注釈は短い独自表現にし、本文を転載しないでください。
- 公表されていない性能や運用を推測しないでください。
- リアルタイムの部隊配置、飛行位置、作戦情報は追加しないでください。

## 情報源の追加

定期巡回したい政府、軍、研究機関、企業の公開ページは`data/sources.json`へ追加します。SNSは新着候補の発見に使えますが、掲載の根拠には一次資料または信頼できる報道を使用します。

## 検査

Pull Requestを作成する前に実行してください。

```bash
python3 tools/build_site.py
python3 tools/verify_wiki.py
```

生成された`docs/`配下の変更もコミットします。
